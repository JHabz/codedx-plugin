This field is required only if the input Analysis Branch is a new branch. This field specifies which parent branch the new branch should be created from.

You can use build/environment variables to construct a different name for each analysis. For a git project on branch 

fix-parameters
: 
${GIT_BRANCH}
would start an analysis for Code Dx branch.

If the entered branch already exists, then this field will be ignored but will log a warning.
